# Week 4 Progress Report: Motion Control Optimization and CAD Learning

## Overview

-Week 4 focused on improving machine motion control, refining the control circuitry, and investigating image generation accuracy. Modifications were made to the existing schematic to improve signal integrity, resulting in better Y-axis motor performance. In parallel, learning and practicing CAD design using Fusion 360 was initiated to support future mechanical development activities.

## Tasks Completed

- Updated and optimized the control schematic for improved system reliability.
- Added a 1 kΩ resistor to the Y-axis STEP/DIR signal lines to improve signal stability and reduce motor oscillation.
- Tested the machine control software and laser module developed by the software team.
- Successfully generated and engraved basic geometric shapes using the laser module.
- Performed step-by-step motor calibration to improve positional accuracy.
- Migrated testing from the ESP32 setup to an Arduino and CNC Shield configuration for comparative evaluation.
- Designed a bottle model and a robotic gripper in Fusion 360 as part of CAD learning and practice.
- Assembled and tested twelve soldering iron units.

## Challenges Faced

- Resolved the Y-axis oscillation issue; however, motor speed and performance required further optimization.
- Observed elongated and distorted engraved images during laser testing.
- Identified positional differences between G-code generated by the software team and LaserGRBL.
- Fine PCB traces were not reproduced accurately during the engraving process, resulting in missing features.

## Key Outcomes

- Improved Y-axis motion stability through circuit-level modifications.
- Achieved successful laser engraving of basic test patterns.
- Enhanced machine calibration accuracy and gained practical experience with CNC-based control systems.
- Developed foundational CAD modeling skills using Fusion 360.

## Next Steps

- Improve image processing and engraving accuracy using the Arduino and CNC Shield setup.
- Implement and validate homing functionality within the CNC control system.
- Continue developing CAD modeling proficiency in Fusion 360.
- Investigate and resolve image scaling and trace reproduction issues.
